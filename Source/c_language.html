C语言高频面经(一）

1. C语言的基本数据类型

C语言主要有以下几种基本数据类型：

  • int：整型，通常占用4个字节。
  • float：单精度浮点型，通常占用4个字节。
  • double：双精度浮点型，通常占用8个字节。
  • char：字符型，通常占用1个字节。

2. C语言中的指针

指针是C语言的一个重要特性，它是存储变量地址的变量。使用指针可以直接访问和修改内存中的数据。

示例代码：


#include <stdio.h>

int main() {
    int a = 10;
    int *p = &a; // p指向a的地址
    printf("a的值: %d\n", *p); // 通过指针访问a的值
    return 0;
}

3. C语言的内存管理

C语言使用动态内存分配函数来管理内存，常用的函数包括：

  • malloc()：分配指定大小的内存。
  • calloc()：分配指定数量的内存块，并初始化为0。
  • realloc()：重新分配已分配的内存块的大小。
  • free()：释放之前分配的内存。
